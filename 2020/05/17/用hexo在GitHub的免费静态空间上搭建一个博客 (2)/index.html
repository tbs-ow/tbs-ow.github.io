用hexo在GitHub的免费静态空间上搭建一个博客

  1. 先下载github和nodejs
  2. 使用npm下载hexo
  3. 然后新建一个文件夹，右键选择git base here然后就会出现
  4. 然后输入npm install -g hexo-cli
  5. 安装成功后输入hexo init然后看见start blogging with hexo就可以输入hexo s
  6. 然后浏览器输入localhost:4000就发现了自己的博客
  7. 注册github账号建立仓库
